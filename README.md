## 如何运行（开发环境）

1. 删除`node_modules`文件夹（如果存在）

2. 清除缓存

   ```
   npm cache clear --force
   ```

3. 安装依赖

   ```
   npm install cnmp
   cnpm install
   ```

4. 运行（8000端口）

   ```
   npm run serve
   ```

## 部署（GitHub page）

1. build项目

   ```
   npm run build
   ```

2. 将获得的`docs`文件夹中文件上传到你的github page仓库
